PWEB-MVC
========
Rewritten pweb CMS using MVC architecture

This is work in progress.

INSTALLATION
============

* Edit the mysql credentials in /includes/init.php;
* Copy/paste to your server target folder;
* Run mvc_pweb.sql in phpmyadmin.
